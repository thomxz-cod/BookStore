# iniciar
- npm init -y


# Baixar tudo de vez
- **npm i ...**


- ``` EX: npm i express moongoose dotenv jsonwebtoken cloudinary bcryptjs cors ```




## express  
- Para melhorar a api

## moongoose
- Fazer conexão com o banco de dados BackEnd

## dotenv
- Guardar informação sigilosa

## jsonwebtoken
- para trabalhar com autenticação 

## cloudinary
- salvar imagem de **fato**

## bcryptjs
- ver os dados "embaralhados"

## cors
- para evitar bugs
